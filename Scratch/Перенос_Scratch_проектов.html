Перенос Scratch-проектов|Руководство Если вы Попали сюда, вы, возможно, хотите конвертировать свою scratch-игру Формат проекта Scratch (. sb, sb2 или. sb3)может быть прочитан только редактором Scratch. Компиляция скретч-проекта может преобразовать его в другой формат, который можно открыть без Скретча. Компиляция была популярным предложением для Scratch 2.0, однако она не была добавлена в обычные онлайн-или оффлайн-Редакторы. Чтобы обойти это, многие Скретчеры придумали способы превратить скретч-проекты в более широко известные форматы, такие как исполняемые файлы Windows Портирование SB3 К сожалению, проекты. sb3 нельзя портировать без их преобразования. Для того чтобы портировать проекты. sb3, их нужно преобразовать в. sb2 .Скачайте Конвертер. здесь Запустите sb3tosb2.py (требуется Python) для запуска конвертера. После сохранения файла. sb2 можно следовать приведенным ниже инструкциям. Загрузите с Scratch редактора 2.0. Зайдите на этот сайт и скачайте конвертер Откройте Zip и распакуйте его. Откройте приложение: для этого кликните по нему правой кнопкой мыши и нажмите открыть с помощью, но сначала скачайте конвертер SWF TO EXE здесь Вернёмся к Конвертеру Sb2 to swf. Откройте приложение: для этого кликните по нему правой кнопкой мыши и нажмите открыть с помощью и найдите в одной из папок конвертера SWF to EXE приложение SA.exe И запустите с его помощью. Отконвертируйте ваш проект в Вашпроектscratch.swf Затем запустите Конвертер SWF TO EXE и нажмите Open Project и выберите ваш отконвертированный в SWF проект, затем нажмите Save Project и Convert Готово! Теперь у вас есть автономная программа!
